validateSection :: [Cell] -> Bool Source #

For a Cell section (i.e., row, column, or box), returns True if the section is valid, False otherwise.

Works by filtering the [Cell] into [[Cell]] where each list contains the list of all cells with realValue \(r \in {1,2,\ldots 9}\). After that we test that each list has length < 2.
